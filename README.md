# Gambit - EEG-Based Video Game Control Using SSVEP
## Description
Welcome to our senior capstone project repository! Here, we employ machine learning to transform EEG measurements into actionable video game commands via Steady-State Visually Evoked Potentials (SSVEP). Our journey begins with crafting a bespoke ML algorithm, then optimizing its performance, followed by its deployment on a microcontroller. The endgame? To empower players with a headband that lets them control a game using just their brainwaves. Dive in to explore our code, findings, and the technology behind this exciting venture.


## Project Overview
1. **EEG Data Collection:** Gather and preprocess raw EEG data to detect SSVEP signals.
1. **Machine Learning Model:** Design, train, and test a model to classify SSVEP signals into game commands.
1. **Optimization:** Refine the model's efficiency to make it suitable for microcontroller deployment.
1. **Microcontroller** Deployment: Integrate the model into a microcontroller ensuring real-time, low-latency game control.
1. **Game Interface:** Let user control a game that responds to the EEG-driven commands delivered by our system.
